Hànyǔ Practice

Welcome to the Hànyǔ practice game. This is a simple and quick way to test and improve your knowledge of Hànyǔ Chinese.


To begin, insert below the file containing the vocabulary you wish to test yourself. Ensure that the file is in csv formatting, and that each line follows the format: "Hanzi, pinyin, english translation". Lines with more or less than three values will not be processed. Also, only use commas in the file if it's to delimit the data. Otherwise, the file may not be parsed correctly.


Attached to the repository is a sample file, comprised of most HSK1 vocabulary words, for your usage and reference. Feel free to modify it as fit to include any words you want.
